# Visual Product Matcher

## Setup

### Server
cd server
npm install
Create .env:
OPENAI_API_KEY=your_key
npm run dev

### Client
cd client
npm install
Create .env:
VITE_API_URL=http://localhost:5000
npm run dev

## Deployment
- Deploy frontend to Vercel
- Deploy backend to Render
- Set environment variables
